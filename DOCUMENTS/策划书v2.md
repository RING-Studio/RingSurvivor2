## 1. 污染值与时间推进（v2口径）

### 1.1 基础规则
- 时间段：早/午/晚（每次任务或不出击都会推进一个时间段）
- 不进行任务度过一个时间段：`污染 += 300`

### 1.2 任务结果的污染变动（默认）
- **任务完成（胜利）**：`污染 = floor(污染 * 0.9) + 300`
- **任务失败**：`污染 += 300`
- **不出击**：`污染 += 300`

说明：
- `floor` 为向下取整，用于保持污染值为整数。
- 不同任务/关卡可拥有独特的污染变动公式，若有特殊任务规则，覆盖以上默认逻辑。

## 2. 任务选择 / MAP 界面（草案）

### 2.1 入口与流程
- 入口：基地出口（`MilitaryCamp`）→ 进入任务选择/MAP界面
- 选择任务 → 确认出击 → 进入关卡
- 未选择任务时不允许出击（当前实现仍为占位条件，后续替换）
- 夜晚是否可出击由任务决定；特殊任务可消耗 2 个时间段

### 2.2 MAP界面布局（草案）
- 顶部信息区：天数、时间段、全局污染值、污染等级提示、额外信息（如天气/地形）
- 中央地图区：可点击的任务节点/区域
- 右侧任务详情区：
  - 任务名称/类型（自由出击/剧情/支线/日常）
  - 任务区域污染基准（基础污染）
  - 任务消耗时间段（默认 1）
  - 预期奖励（金币/解锁/材料）
  - 污染变动预估（默认显示 `污染=round(污染*0.9)+300`）
  - 推荐战力/难度提示
- 底部按钮区：开始任务 / 返回基地

### 2.3 任务状态与筛选
- 状态：已解锁 / 未解锁 / 已完成
- 任务节点可标注“本日可执行次数”或“推荐时间段”
- 可选筛选：剧情任务、收益任务、风险任务

### 2.4 数据字段建议（供后续实现）
- `mission_id` / `mission_name`
- `mission_type`（story/side/free）
- `base_pollution`
- `time_cost`（默认 1）
- `reward_preview`
- `pollution_formula`（默认：victory=0.9, defeat=1.0）
- `unlock_condition`
- `recommended_power`

### 2.5 已确认事项

- 特殊任务允许消耗 2 个时间段
- MAP 界面需要显示额外信息（如天气/地形提示）
