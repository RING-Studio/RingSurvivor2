# 新升级选项（命名重构 — 定稿）

更新日期：2026-02-11  
目的：将现有"偏口语/偏直白"的升级命名升级为更专业、辨识度更高的术语。  
原则：
- 好的名字不改，不为改而改
- 基础/白色升级用清晰简明的名词，不搞深奥术语
- 中高品质升级可以用更专业、更有感觉的词
- 优先使用真实存在的军事/工程术语
- 注意车辆题材适配：避免人体/刀刃等与车辆无关的意象
- ID 可随名称一起修改（参见 `注意事项.md`：不考虑存档兼容）

> **补充设定**：玩家为车辆，敌方涵盖生物（变异生物、人）和机械（车辆为主）。  
> "流血"机制仅对生物生效（机械流血抗性 = INF）。破片系列强化对机械和生物应有不同表现。
