## 1. 污染值系统（来自 `DOCUMENTS/策划书.md`）

### 1.1 基础规则（旧）
- 污染值区间：良好 <3000、轻度 <6000、中度 <8000、重度 <10000、爆表 ≥10000
- 初始值：7000
- 基础日增长：900/天（早/午/晚各增长 300）
- 降低方式：击杀污染怪

### 1.2 区域污染与全局污染（旧）
- 任务初始区域污染值 = （任务基础污染 + 全局污染值） / 2
- 区域污染影响：
  - 强度 = 怪物基础污染 * sqrt(区域污染值 / 7000)
  - 出怪频率 = 基础出怪频率 * (区域污染值 / 7000)
- 区域污染随时间提升：约 1.0 / 秒
- 任务完成后，**全局污染值变动 = 区域污染值变动 / 2**
  - 全局随时间增长的 +300 另算

## 2. 时间消耗与出击限制（旧）

来自 `DOCUMENTS/策划书.md`：
- 每天 早/午/晚 三个单位时间
- 每次执行任务默认消耗一单位时间
- 任务/出击只能在早上或下午
